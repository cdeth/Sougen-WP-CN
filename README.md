## 介绍
_Sougen 白皮书 （最近一次更新时间：2022年2月）；简体中文版由 cryptodesigner.eth 翻译，由 k.mirror.xyz 和 Alice Zou 校对，如您对中文白皮书内容有任何建议，可通过 Twitter 联系译者（@cd_eth）。_



Sougen是一个去中心化的元宇宙虚拟世界，人们能够以新的方式在此进行社交、学习、工作和娱乐活动。  
我们的3D元宇宙为社区提供了一个场所，让人们能够以更加身临其境的方式聚会和共享时光，为他们的体验增加了一种物理感受。  
我们相信元宇宙应该从集体努力中成长。这就是为什么我们的用户将被授予创作工具的使用权，用户能够在 Sougen 中无缝创造自己的体验，并在Web 3.0的帮助下获得他们对创作的全部所有权。  
此外，我们的主要愿景是元宇宙应该向所有人开放。  
这就是为什么Sougen被构想为一个 Web 优先的元宇宙，用户可以从任何地方和任何设备上通过浏览器访问它。  
我们始终专注于打造一个用户友好、包容和无障碍的用户体验，同时也在努力打造有史以来最沉浸的3D 元宇宙体验。  
我们也将为社区提供源源不断的机遇。

这就是为什么Sougen团队在幕后工作了几个月，构建了Sougen 世界的基础。作为概念的验证，团队还专注于精心制作第一个微宇宙体验：序幕（The prologue）。  
序幕将展示我们的技术专长，并将作为我们未来为用户提供的一个示例。

## GEN 世界 （GEN World）
GEN 世界是一个所有社区聚集在一起进行创造和交流的地方。  
GEN地图将由102,400 份土地组成，所有这些土地都与NFT 连接，用户可通过逐次的土地销售获得。其中，8192份土地（8%）将由 Sougen 团队持有，以创造免费和高质量的体验；11264份土地（11%）将作为储备，作为奖励分发给合作伙伴、创作者和用户。  
土地所有者将获得各种好处：例如对其土地的完全所有权，能够自由分享、出租或出售他们的土地，或者享有每月赚取 Sougen 代币和 NFT 掉落奖励的能力。  
Sougen 编辑器将允许土地所有者在他们的土地上创造自己的体验：可以是创业、做广告、做游戏，或者是展示他们的 NFT 收藏。土地所有者将可以免费使用 Sougen 编辑器的全部高级功能。  
GEN 世界还将有一个使命：即成为一个展示最佳微宇宙的门户，游客们可以一键访问并体验。
